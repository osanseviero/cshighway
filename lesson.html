Tipografía y recursos de estilo

Tipografía

Para modtrar nuevas secciones y subsecciones, se pueden usar los tags <h2> hasta <h6>. Además se debe poner cada línea dentro de un <p>. Recuerda hacer saltos de líneas con <br>.

Hay muchas formas de resaltar texto.

  • negritas con el tag de HTML <strong>.
  • cursiva con el tag <em>.
  • marcar con el tag <mark>.
  • borrar con el tag <del>.
  • rayado con el tag <s>.
  • texto insertado con el tag <ins>.
  • texto subrayado con el tag <u>.
  • incluso poner el texto chiquito para cosas poco importantes con el tag <small>.
